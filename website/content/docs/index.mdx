## 为什么选择better-react?

- **增强开发体验**：提供更简洁的API和更好的类型支持
- **高性能**：优化的组件实现，确保良好的性能表现
- **易于使用**：与React生态系统完美集成

## 安装

```bash
npm install better-react
# 或
yarn add better-react
```

## 使用示例

```jsx
import { EnhancedComponent } from 'better-react';

function App() {
  return <EnhancedComponent />;
}
```
