# better-react-dom-helper

DOM-specific helper utilities for better-react framework.

## Installation

```bash
npm install better-react-dom-helper
# or
pnpm add better-react-dom-helper
```

## Peer Dependencies

- `better-react` (workspace dependency)
- `better-react-dom` (workspace dependency)
- `better-react-helper` (workspace dependency)
- `wy-dom-helper` (workspace dependency)
- `wy-helper` (workspace dependency)

## Features

- **DOM Utilities** - Enhanced DOM manipulation helpers
- **Animation Support** - Animation utilities for DOM elements
- **Form Helpers** - Form handling utilities
- **Event Management** - Advanced event handling
- **Accessibility** - A11y helper functions

## Usage

```ts
import {} from /* DOM helper utilities */ 'better-react-dom-helper'

// DOM-specific utilities for better-react
```
